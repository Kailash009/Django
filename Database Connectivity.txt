Now, Perform Migration Commands:-
---------------------------------------------------
1. python manage.py makemigrations  -> This command see your model class and generate ORM Query.
2. python manage.py migrate -> This command see your migration file and generate tables into your database.
It will give you 12 tables.
11  -> Predefined apps.
1  -> it is your model class table.
