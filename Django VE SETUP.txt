Virtual Enviornment ->  VE provide isolation for your project.
-------------------------------------------------------------------------------------------------------------------------------
Steps to create VE for Your Project:-
---------------------------------------------------
1, Create one folder name as -> Batch 10 AMdjango -> another folder -> myallprojects -> inside create another project 1-> project-1 -> open this project1 path with CMD.
2. First of all, Install VE 
         		pip install virtualenv
How to check virtualenv version -> virtualenv --version
3. Create VE:-
          virtualenv anyFolderName 
           virtualenv venv
4. Activate your VE:-
         virtualenvFoldername/scripts/activate
	venv\scripts\activate
5. Now check whether your project has connected Locally or Globally -
	pip freeze
6. Command to disconnect your project with VE-
     >> deactivate
